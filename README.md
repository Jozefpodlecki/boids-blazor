# Flocking Simulator 

## Getting Started

```bash
git clone https://github.com/Jozefpodlecki/boids-blazor
cd boids-blazor
dotnet restore
dotnet run
```

## Build

```bash
dotnet publish --configuration Release --output build
```

## Tailwind

```
npx @tailwindcss/cli -i ./wwwroot/css/styles.css -o ./wwwroot/css/output.css --watch
```
